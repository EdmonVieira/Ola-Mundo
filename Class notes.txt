Aula 1
O que é VCS? Version Control Software
O que é controle de versão de software? Prática de rastrear e gerenciar alterações de controle de versões por meio de numerações diferentes.
O que é Git? Software para controle de versão de arquivos localmente. Repositório Local.

Aula 2
O que é GitHub?
- Plataforma de rede social para programadores.
- Repositório Remoto com hospedagem de código fonte.
- Issues, função colaborativa para detecção de problemas no código.
- Forks, função criativa para alterar um código sem alterar a versão original.

Aula 3
Quem veio primeiro? Git
História
1985 - CVS (Centralizado, Open Source, Popular e Problemático)
2000 - SVN (Centralizado, Open Source, Ainda ativo, CVS like)
2000 - BitKeeper (Distribuido, Proprietário, Versão Comunidade, CVS free)
2004 - SourcePuller (Engenharia Reversa, Recursos destravados
2005 - Git (Distribuido, Open Source, Feito em 10 dias, Performance)
2008 - Github (Proprietário, hospedagem de código, baseado em Git)
2018 - Github foi coprado pela Microsoft

Aula Extra 3
HTML é uma linguagem de marcação
Guia da Linguagem Markdown (extensão .md) - https://www.markdownguide.org/
- # ou = - Título
- ## ou == - Título 1
- --- ou *** - Linha de demarcação
- **x** ou __x__ - Negrito
- *x* ou _x_- Itálico
- __*x*__ - Negrito e Itálico
- 1. - cria lista numerada
-  1. - cria sublista numerada
- * - cria lista demarcada
-  * - cria sublista demarcada
- - [ ] - cria lista de tarefa
- - [x] - cria lista de tarefa feita
- ![xxx](https://xxx) - carrega imagens em .png,.jpeg
- [xxx](https://xxx) - carrega link
- Col1 | Col2 | Col3 - cria cabeçalho de tabela
- ---|---|--- - cria tabela
- xxx | xxx | xxx - insere dados na tabela
- `x` - insere o texto de modo monoespaçado com fundo cinza
- ```
  num = int(input('Digite um valor:')
  if num % 2 == 0:
    print(f'O valor {num}) é PAR')
  else:
    print(f'O valor {num} é ÍMPAR')
  ``` - insere o texto em formato de código
- :vulcan_salute - insere emoticon
- > - insere um reply
- @ - faz menção a alguém
